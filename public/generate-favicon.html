Generate Favicon with Calegar Font

This tool generates favicon files using the Calegar font. Open this file in your browser after starting the Next.js dev server.

Instructions:

  1. Make sure your Next.js dev server is running (npm run dev)
  2. Open this file in your browser: http://localhost:3000/generate-favicon.html
  3. Click the download buttons to save the favicon files
  4. Move icon.png to the app folder (it will replace the SVG)
  5. Next.js will automatically use app/icon.png as the favicon
